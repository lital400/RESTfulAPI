API Reference Index


Method URL Properties Description Comments
GET / N/A Help Page Any undefined URLs will be redirect to the Help Page
GET /Users N/A Retrieve a list of all users with all their properties A list of users is displayed in JSON format
POST /Users UserID, Fist Name, Last Name, Email Address, Password Create a new User record The User Id must be unique
If Id is not unique, a 400 error (Bad Request) is returned
GET /Users/{ID} N/A Retrieve a specific user with all its properties (if it exists) Provide the User Id in the URL.
If Id is not found, a 404 error (Not Found) is returned
PATCH /Users/{ID} Fist Name, Last Name, Email Address, Password Update (change) any data for an existing User Cannot update the User Id.
May update any or all other user properties
DELETE /Users/{ID} N/A Delete a user (if it exists) Delete a user by providing the User Id in the URL
